# xcode-snippets

My Xcode Snippets

## Installation

Copy files to `~/Library/Developer/Xcode/UserData/CodeSnippets`

## Cheat Sheet

Command  | Description
------------- | -------------
`logm`|Log a method call
`mark`|Add a mark pragma
`pret`|Log a function's name
`prop`|Create a nonatomic property
`stri`|Convert an integer to a string
